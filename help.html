Guidance

Here is the simple guidance of how to use this website. To view database of every bars in Corvallis, simply click on database and then bars. The bar page will display all the information about each bar. If you wish to look at only specific detail, simply enter the detail in the search bar.

To Write a review, simply click on write a review. In order to write a review, user must input the valid bar id and name. Afterwards user can write or update any information about a certain bar as desired. The information will be updated in our database and display it when visit the database page.
